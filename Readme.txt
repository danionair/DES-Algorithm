This program take a cryptogram and password in hexadecimal format of 64 bit each one and it return the keys, blocks and deciphered text in binary, hexadecimal, decimal and ascii format. 

You can try it with the next examples:

Cryptogram: 09ab068d5cb9ce21
Password: 5914D1EC7703735C
deciphered text: ravelled

Cryptogram: 12088ba3e74cabde
Password: 70617373776F7264
